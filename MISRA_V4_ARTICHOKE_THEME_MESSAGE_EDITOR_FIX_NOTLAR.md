# Dyt. Mısra Çakır V4 - Artichoke Theme + Message/Admin Fix

Bu sürümde kullanıcının yeni gönderdiği enginar/sage/plum paletine göre tüm ana UI yüzeyleri yeniden renklendirildi.

## Renk Paleti
- Mist: #DEE2EE
- Lavender: #A78FA9
- Plum: #654258
- Dark Sage: #516559
- Sage: #A0B18F
- Stone: #D0D0CE

## Yapılanlar
- Navbar, sidebar, footer, admin panel, formlar, butonlar, kartlar, blog kartları ve editor alanları yeni palete uyarlandı.
- Sağdaki sosyal barda Instagram / Mail / TikTok / Yukarı çık düzeni korundu.
- Footer'da WhatsApp ve X kaldırıldı; Instagram / E-posta / TikTok bırakıldı.
- İletişim kartları 2x2 kompakt düzene geri çekildi ve uzun e-posta taşması engellendi.
- Admin randevu/mesaj sayfası tablo yerine açılır kart yapısına çevrildi.
- Açılır randevu kartında tam mesaj, kişi bilgileri, telefon, e-posta, hedef, randevu zamanı görünür.
- WhatsApp ile yanıtlamaya ek olarak Mail ile yanıtla butonu eklendi.
- Blog ekle/düzenle sayfasında içerik editörünün başlık alanının odağını çalma hatası düzeltildi.
- Font ve boyut seçimlerinde Quill selection korunarak yazı bozulma ihtimali azaltıldı.
- CSP güvenlik başlığı eklendi ve Quill CDN/font kaynaklarıyla uyumlu hale getirildi.
